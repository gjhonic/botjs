#Я Бот 
И вот мои возможности:
```
- !say [Сообщение] - Повторю сообщение
- !clear [Кол-во] - Удалю n сообщений
- !help - Покажу список команд
- !link - Покажу как меня установить
(ВНИМАНИЕ СЛЕДУЮЩИЕ КОМАНДЫ ТЕСТИРУЮТСЯ)
- !calc [число][операция][число]... - Решу пример 
- !ex выполню код (JS)
```
Подробнее о команде !ex:
```
Синтаксис: (' -> `)
!ex '''js
>>>
[Код который нужно выполнить]
'''
```
