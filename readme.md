# Discord Bot JS

Bot for programmers
v 0.1.2

**Instruction manual**
1) Loading
```
git clone https://github.com/gjhonic/botjs.git
```
2) Install package
```
npm install
```
3) Create file config.json and insert code
```json
{
    "token" : "token",
    "prefix" : "prefix"
}
```
4) Start bot
```
node start.js
```
