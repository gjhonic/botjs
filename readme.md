# Discord Bot from JS

**Instruction manual**

1) Loading
```
git clone https://github.com/gjhonic/botjs.git
```

2) Install package
```
npm i discord.js
npm i osmosis
```

3) Create file auth_data.js and insert code
```js
class MyAuth{
  login = "<login>";
}
module.exports = MyAuth;
```

4) Start bot
```
node bot.js
```
